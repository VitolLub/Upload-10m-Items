Settings

Zoom

This is used to shrink or expand The Camelizer's main window. If you simply want to get rid of scrollbars on the window, find your browser's zoom setting and set this box to match. For example: if your browser is set to 150% zoom, select the entry in this list labeled "(browser 150%)".

To help avoid potential over- or under-zooming, please only move one zoom level at a time. If scrollbars remain after changing this setting, close and re-open The Camelizer window.
